# ProyectoSIGED

El Sistema de Gestión Académica es una aplicación orientada a la administración eficiente de información académica. Este sistema busca optimizar la toma de decisiones mediante el manejo adecuado de datos, asegurando su almacenamiento, uso responsable y análisis.
--
El proyecto incorpora principios clave como:
- Gestión responsable de datos personales.
- Aplicación del principio de responsabilidad única en el desarrollo del sistema.

# Arquitectura Backend SIGED

Esta carpeta contiene los reportes arquitectonicos de los modulos backend
implementados o auditados en Django REST Framework.

## Modulos documentados

- `actores_academicos/actores_academicos_reporte.qmd`
- `institucion/institucion_reporte.qmd`
- `planificacion/planificacion_reporte.qmd`
- `ubicacion/ubicacion_reporte.qmd`

## Criterio de organizacion

Cada carpeta representa un modulo funcional del backend en `sga/apps/`. Los
reportes describen inventario, endpoints, capas, riesgos y avance real del
modulo.
